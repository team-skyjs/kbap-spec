> 핵심 가치(안 먹는 것 거르기)가 꺼진 상태 — 디스커버리(7/21) 후 예진 확정.

## 결정

1. **재개 모달은 현행 유지** — 미완자에게만. 스킵-완료자는 의도적 선택으로 존중, 반복 모달 금지 (Q-09 종결)
2. **출시 전 1개만**: 빈 프로필 사용자의 **첫 스캔 결과 상단 1줄 배너** — "기피 재료를 설정하면 이 메뉴판에서 피할 걸 짚어드려요" + 설정 링크. 가치 증명 순간(스캔 직후)이 설득 타이밍이라는 가설. 비차단·기존 데이터만(KB-208/P-038)
3. **나머지는 측정 후**: Amplitude(KB-161)로 스킵률·빈 프로필 유지율 확인 뒤 순차 결정
