## 🎯 크리티컬 패스 (출시까지 무엇이 무엇을 막나)

```
BE 나머지 엔드포인트 배포 ─┐
BE Swagger(현재 2개) ──────┤
                          ▼
                    KB-66 API 공통 레이어 ──▶ KB-67~75 화면별 실연결 ──▶ 실기기 QA(안전검증) ──▶ 제출
디자인 롱폴(인트로/잠금/뱃지) ──▶ KB-76·78·84·25 UI ──────────────────────┘
```

1. **BE 엔드포인트 미배포가 최대 병목.** 현재 Swagger엔 스캔·상세 2개만. 인증·프로필·홈·검색·리뷰·랭킹 계약 미정의 → 실연결 다수 블록.
2. **KB-66(공통 레이어)** = 실연결 9개의 선행. 응답 래퍼 `{success,payload,message}` 확정으로 착수 가능(진행 중).
3. **디자인 롱폴** = 인트로 3슬라이드+일러스트, 비회원 잠금(상세/리뷰), 위험도 4-state 뱃지. 발주 완료(2026-07-06), 수령 후 UI 착수.
4. **안전 검증(SC-003 false-safe=0)** = BE 실위험도 붙은 뒤 실기기 QA. 출시 직전 관문.
